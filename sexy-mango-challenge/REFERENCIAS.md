### Apoyo de Inteligencia Artificial

Se utilizó **Claude Code (Anthropic)** como asistente durante el desarrollo del proyecto para:
- Generar la estructura inicial del proyecto Angular con todos los componentes.
- Escribir el código base de los servicios `JuegoService` y `AudioService`.
- Asistir con la implementación de Angular Signals (`signal`, `computed`, `effect`).
- Implementar la animación CSS 3D de flip de cartas.
- Implementar el temporizador circular en SVG.
- Generar los efectos de sonido mediante la Web Audio API.
- Aplicar la identidad visual de marca (colores, tipografía, layout responsivo).

Todo el código fue revisado y comprendido por el estudiante. El uso de IA quedó documentado aquí según los lineamientos del curso.
